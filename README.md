# LeetCode Daily Questions
A simple repository to track and solve LeetCode Daily Questions consistently. This project helps maintain coding discipline, improve problem-solving skills, and build strong Data Structures & Algorithms (DSA) fundamentals.

##About
This repository contains solutions for LeetCode Daily Challenges organized by date.
Each solution includes:

Problem name
Difficulty level
Date
Source code

The goal is to:
Practice DSA regularly
Improve coding interview preparation
Maintain consistency through daily problem solving
